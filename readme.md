This is a beginer application using the mern stack in order to better understand how to code express backends and interact with mongoDB

## technologies used ->

1. Node
2. ExpressJs
3. JWT
4. React
5. Bcrypt
6. Config
7. Express-Validator
8. Gravatar
9. axios
10. normalize-url

React is done using primarily hooks
